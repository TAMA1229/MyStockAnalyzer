# 📈 K-Stock Insight (프리미엄 주식 포트폴리오 & 시세 분석 대시보드)

> **한국 및 해외(미국) 주식 통합 시세 관리, 이동평균법 기반 매매 및 실현손익 자동 연산, 예수금 잔고 관리, 가족 통합 자산 분석, 텔레그램 실시간 푸시 알림 및 AI 투자 전략 제언 서비스**

---

## 🌟 주요 기능 (Key Features)

### 1. 🇰🇷 국내 & 🇺🇸 해외(미국) 주식/ETF 통합 시세 관리
- **실시간 환율 연동**: 네이버 금융 기반 실시간 USD/KRW 환율(10분 자동 캐싱) 연동으로 해외 주식의 원화 평가액 자동 계산.
- **다양한 차트 지원**: 일봉, 캔들 일봉, 3개월 면적, 1년 변동 차트 제공.
- **스마트 검색**: 국내/해외 종목명, 종목코드, 영문 티커 및 한글 칭호 검색 자동완성 지원.

### 2. 📊 이동평균법 기반 매매 기록 & 실현 손익 자동 연산
- **매수/매도(BUY/SELL) 관리**: 가중평균 평단가(`avgPrice`), 보유 수량(`totalQty`), 그리고 매도에 따른 **누적 실현 손익(Realized Profit)** 자동 집계.
- **유효성 검증**: 날짜별 보유 수량을 체크하여 초과 매도 시도를 사전에 차단.
- **손익 분리 표기**: 포트폴리오 종합 손익, 미실현 평가 손익, 확정 실현 손익을 명확히 분리하여 디스플레이.

### 3. 💵 예수금(Cash) 잔고 자동 연동
- **자동 가감 시스템**: 매수/매도 등록 및 삭제 시 거래 금액(외화는 매매 시점 환율 반영)이 예수금 잔고에 자동 반영.
- **수동 잔고 보정**: 입출금 등에 따른 현금 잔고 수동 수정 폼 제공.
- **총자산 공식 개편**: `총 자산 = 총 주식 평가액 + 예수금(현금)` / `순자산 = 총 자산 - 부채`.

### 4. 👪 가족 통합 자산 분석 대시보드 (Admin View)
- **통합 자산 관리**: 가족 구성원(아빠, 엄마, 딸 등)의 포트폴리오를 한눈에 모니터링.
- **실시간 지분율 차트**: 구성원별 자산 기여도 수평 바 그래프 및 주요 보유 종목 Top 5 집계.
- **통합 ROE 산출**: 가족 전체의 주식 평가액, 부채, 예수금 및 누적 실현손익 통합 계산.

### 5. 🔔 텔레그램(Telegram) 실시간 모니터링 엔진
- **1분 주기 크론 감시**:
  - **지정 상/하한가 도달 알림**: 목표가 돌파 시 즉시 메신저 푸시 전송.
  - **당일 시세 급등락 알림**: ±5.0% 이상 시세 변동 시 감지.
  - **지수 대비 추가 하락 감지**: KOSPI / NASDAQ 지수 변동률 대비 설정 비율 이상 초과 하락 시 감지.
- **세이프가드 필터**: 장마감(`CLOSE`), 휴장(`HOLIDAY`) 시 알림 전송을 차단하여 불필요한 알림 방지 및 당일 중복 발송 방지.

### 6. 📰 Google News RSS 기반 감성 분석 & AI 투자 전략
- **뉴스 파싱 및 감성 지수화**: 기업 관련 최신 기사의 호재/악재 키워드를 분석하여 뉴스 여론 점수화.
- **투자 제언 카드**: 밸류에이션(PER, PBR, 배당수익률), 뉴스 감성, 가격 모멘텀을 가중 조합하여 5단계 투자 의견(적극 매수~적극 매도) 및 매력/위험 요인 브리핑.

### 7. 📱 모바일 UX 최적화 & 수동 새로고침
- **반응형 대시보드**: 모바일 환경에서 글로벌 지수(KOSPI, NASDAQ) 및 텍스트 짤림 방지(`flex-wrap`, `white-space: nowrap`).
- **원클릭 새로고침**: 실시간 환율 및 글로벌 시장 브리핑 카드 스피너 애니메이션 수동 새로고침 지원.

## 🛠 기술 스택 (Tech Stack)

- **Backend**: Node.js, Express.js (Native ES / CommonJS fetch)
- **Frontend**: HTML5, Vanilla CSS3 (Glassmorphism, CSS Variables, Responsive Layout), JavaScript (ES6+)
- **Icons**: Lucide Icons CDN
- **Database / Storage**: Local JSON File Persistence (`/data` Volume Target)
- **Containerization**: Docker, Docker-Compose

## 🚀 시작하기 (Getting Started)

### 1. 로컬 환경 실행

```bash
# 레포지토리 클론
git clone <your-repository-url>
cd korean-stock-analyzer

# 의존성 패키지 설치
npm install

# 서버 가동
npm start
# 또는 개발 모드 (자동 감시)
npm run dev
```
접속 주소: `http://localhost:3000`

### 2. Docker를 이용한 배포

```bash
# 도커 빌드 및 컨테이너 실행
docker-compose up -d --build
```
